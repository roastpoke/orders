# 客户管理系统 - 销售抢单记录平台

一个简洁高效的客户管理系统，专为销售团队设计，支持客户录入、归属确权、跟进记录等功能。

## 📱 功能特点

### 核心功能
- ✅ **客户管理** - 客户信息录入、编辑、删除
- ✅ **归属确权** - 首次登记时间自动留痕，明确客户归属
- ✅ **权限控制** - 管理员/员工分级权限
- ✅ **跟进记录** - 记录每次客户沟通情况
- ✅ **数据统计** - 多维度数据分析与排行榜
- ✅ **数据导出** - 支持导出CSV格式

### 用户管理
- ✅ 手机号注册/登录
- ✅ 管理员审批新用户
- ✅ 用户禁用功能

## 🚀 使用方法

### 方式一：直接打开
1. 双击 `index.html` 文件
2. 或在浏览器中打开该文件

### 方式二：本地服务器（推荐）
```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js
npx serve .

# 使用 VS Code Live Server 插件
```
然后访问 http://localhost:8080

## 📋 客户信息字段

### 必填字段
- 客户姓名
- 联系电话
- 客户来源（抖音/小红书/微信/门店/转介绍/电话/其他）

### 选填字段
- 预产期/需求日期
- 预算范围
- 房型偏好
- 客户状态
- 备注信息

### 系统自动生成
- 登记时间（精确到秒）
- 跟进人（当前登录用户）
- 客户编号

## 🔒 权限说明

### 管理员权限
- 查看所有客户
- 编辑/删除任意客户
- 审批新用户注册
- 禁用用户账号
- 查看员工排行榜
- 导出全部数据

### 员工权限
- 添加新客户
- 查看/编辑自己的客户
- 删除自己的客户
- 添加跟进记录
- 导出自己的数据

## 💾 数据存储

当前版本数据存储在浏览器 LocalStorage 中：
- 优点：无需服务器，即开即用
- 注意：清除浏览器数据会丢失记录

如需持久化存储，可升级为：
- Firebase 云数据库
- 自建后端 + MySQL/PostgreSQL
