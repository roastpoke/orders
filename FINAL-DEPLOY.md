## 🔧 部署步骤（详细版）

### 第一步：注册 Leancloud

1. 访问 [leancloud.cn](https://leancloud.cn)
2. 注册账号（手机号即可）
3. 创建应用，选择"开发版"（免费）
4. 记录 AppID 和 AppKey

### 第二步：修改系统配置

在 `js/firebase-config.js` 文件中，替换为 Leancloud 配置：

```javascript
// Leancloud 配置
AV.init({
  appId: "你的AppID",
  appKey: "你的AppKey",
  serverURL: "https://你的域名.lc-cn-n1-shared.com"
});
```

### 第三步：部署网站

**方法1：Vercel部署（推荐）**

```bash
# 1. 安装vercel
npm install -g vercel

# 2. 进入项目目录
cd e:\the

# 3. 部署
vercel
# 按提示操作，不到1分钟完成
```

**方法2：直接拖拽部署**

1. 访问 [vercel.com](https://vercel.com)
2. 登录后点击 "Add New Project"
3. 拖拽 `e:\the` 文件夹到页面
4. 等待30秒完成部署
